Guillaume Florent's Github Pages

Welcome to my GitHub Pages which are about my open source development projects at github.com/guillaume-florent

These projects are about yacht design, aero and hydrodynamics, CAD, navigation ...

The goal is to develop a complete software toolchain for yacht design, with a focus on racing.

The following areas are covered:

  • parametric and algorithmic shape generation
  • hydrostatics
  • 2d foil sections optimization
  • performance prediction
  • race simulation
  • CFD RANS cases generation
